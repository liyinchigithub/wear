# 环境
JDK >=11
SDK >=29


# 编译
```shell
./gradlew clean assembleDebug   
```

# 运行
```shell
./gradlew installDebug
```

# 打包
```shell
./gradlew clean assembleRelease
```


# 打包apk
```shell
./gradlew assembleDebug
```
